User Endpoints

GET /api/users

Get a list of users in the current user's workspace

GET /api/users/me

Get details of the currently logged-in user

GET /api/users/{userId}

Get details of a specific user

PATCH /api/users/me

Update the currently logged-in user's profile

PUT /api/users/me/status

Update the currently logged-in user's status

DELETE /api/users/{userId}

Delete a user (Admin only)

GET /api/users/me/preferences

Get the current user's notification preferences

PATCH /api/users/me/preferences

Update the current user's notification preferences

GET /api/users/{userId}/direct-messages

Get direct messages with a specific user

GET /api/users/me/avatar

Get the current user's AI avatar details

POST /api/users/me/avatar

Create an AI avatar for the current user

PATCH /api/users/me/avatar

Update the current user's AI avatar settings

GET /api/users/me/avatar/history

Get the communication history for the current user's AI avatar

POST /api/ai/feedback

Provide feedback on an AI avatar interaction

POST /api/ai/send-message

Send a message using the AI avatar

Workspace Endpoints

GET /api/workspaces

GET /api/workspaces/{workspaceId}

POST /api/workspaces

Channel Endpoints

GET /api/workspaces/{workspaceId}/channels

Get a list of channels in a workspace

Get details of a specific channel

Create a new channel

Update a channel's details

DELETE /api/channels/{channelId}

Delete a channel

GET /api/users/me/channels

Get a list of channels the current user is a member of

Channel Member Endpoints

POST /api/channels/{channelId}/join

Join a public channel

DELETE /api/channels/{channelId}/leave

Leave a channel

GET /api/channels/{channelId}/members

Get a list of members in a channel

POST /api/channels/{channelId}/members

Add members to a channel

DELETE /api/channels/{channelId}/members/{userId}

Direct Message Endpoints

GET /api/direct-messages/{directMessageId}

Get details of a specific direct message

POST /api/direct-messages/{recipientId}

Send a new direct message to a user

GET /api/users/me/direct-messages

Get a list of the current user's direct message conversations

GET /api/direct-messages/conversations/{userId}

Get messages in a direct message conversation with a specific user

File Endpoints

POST /api/files/upload

Upload a file

Note: This endpoint requires a form-data (multipart) request with a file upload. Testing through this interface is limited. Try using Postman for a complete test with file uploads. 
// Example response:
{
  "fileId": "example-file-id",
  "fileUrl": "https://example.com/files/example.pdf",
  "filename": "example.pdf",
  "mimeType": "application/pdf",
  "size": 1024,
  "uploadedBy": "user-id",
  "createdAt": "2023-06-15T10:30:00Z"
}

GET /api/files/{fileId}

Get file metadata

GET /api/files/{fileId}/download

Download a file

// This endpoint returns the file itself, not JSON.

DELETE /api/files/{fileId}

Delete a file

GET /api/channels/{channelId}/files

Get files shared in a channel
