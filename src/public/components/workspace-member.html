Workspace Member Endpoints

GET /api/workspacemembers

Get all workspace members

GET /api/workspacemembers/workspace/{workspaceId}

Get all members of a specific workspace

GET /api/workspacemembers/user/{userId}

Get all workspace memberships for a specific user

GET /api/workspacemembers/me

Get all my workspace memberships

GET /api/workspacemembers/{id}

Get a specific workspace member by ID

GET /api/workspacemembers/workspace/{workspaceId}/user/{userId}

Get workspace membership by workspace and user IDs

GET /api/workspacemembers/check/{workspaceId}

Check if current user is a member of a workspace

GET /api/workspacemembers/checkadmin/{workspaceId}

Check if current user is an admin of a workspace

POST /api/workspacemembers

Add a member to a workspace

PATCH /api/workspacemembers/{id}

Update a workspace member (e.g., change role)

DELETE /api/workspacemembers/{id}

Remove a member from a workspace

POST /api/workspacemembers/bulk

Bulk add members to workspaces

GET /api/workspacemembers/profile/{workspaceId}

Get my profile in a specific workspace
