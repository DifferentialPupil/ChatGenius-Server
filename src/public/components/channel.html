Channel Endpoints

GET /api/channels/workspace/{workspaceId}

Get a list of channels in a workspace

GET /api/channels/{channelId}

Get details of a specific channel

POST /api/channels/create/{workspaceId}

Create a new channel

PATCH /api/channels/{channelId}

Update a channel's details

DELETE /api/channels/{channelId}

Delete a channel

GET /api/channels/{channelId}/members

Get a list of members in a channel

POST /api/channels/{channelId}/members

Add members to a channel

DELETE /api/channels/{channelId}/members/{userId}

Remove a member from a channel

POST /api/channels/{channelId}/join

Join a public channel

DELETE /api/channels/{channelId}/leave

Leave a channel

GET /api/channels/{channelId}/messages

Get messages for a specific channel

POST /api/channels/{channelId}/messages

Send a new message to a channel

GET /api/channels/{channelId}/pins

Get a list of pinned items in a channel

POST /api/channels/{channelId}/pins

Pin a message or file to a channel

DELETE /api/channels/{channelId}/pins/{pinnedItemId}

Unpin an item from a channel

GET /api/channels/user/me

Get a list of channels the current user is a member of
