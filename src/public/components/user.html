User Endpoints

GET /api/users

Get a list of users in the current user's workspace

GET /api/users/me

Get details of the currently logged-in user

GET /api/users/{userId}

Get details of a specific user

PATCH /api/users/me

Update the currently logged-in user's profile

PUT /api/users/me/status

Update the currently logged-in user's status

You can either enter plain text for status OR a JSON object with statusText and optional expiresAt fields.

DELETE /api/users/{userId}

Delete a user (Admin only)

GET /api/users/me/mentions

Get mentions for the current user

GET /api/users/me/mentions/count

Count unread mentions for the current user

GET /api/users/me/notification-preferences

Get the current user's notification preferences

PATCH /api/users/me/notification-preferences

Update the current user's notification preferences

GET /api/users/me/direct-messages

Get direct messages for the current user

GET /api/users/me/avatar

Get the current user's AI avatar details

POST /api/users/me/avatar

Create an AI avatar for the current user

PATCH /api/users/me/avatar

Update the current user's AI avatar settings

GET /api/users/me/avatar/history

Get the communication history for the current user's AI avatar

POST /api/ai/feedback

Provide feedback on an AI avatar interaction

POST /api/ai/send-message

Send a message using the AI avatar
