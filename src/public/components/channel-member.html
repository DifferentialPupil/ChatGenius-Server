Channel Member Endpoints

GET /api/channelmembers

Get all channel members

GET /api/channelmembers/channel/{channelId}

Get all members of a specific channel

GET /api/channelmembers/user/{userId}

Get all channel memberships for a specific user

GET /api/channelmembers/me

Get all my channel memberships

GET /api/channelmembers/{id}

Get a specific channel member by ID

GET /api/channelmembers/channel/{channelId}/user/{userId}

Get channel membership by channel and user IDs

GET /api/channelmembers/check/{channelId}

Check if current user is a member of a channel

POST /api/channels/{channelId}/join

Join a public channel

DELETE /api/channels/{channelId}/leave

Leave a channel

GET /api/channels/{channelId}/members

Get a list of members in a channel

POST /api/channels/{channelId}/members

Add members to a channel

PATCH /api/channelmembers/{id}

Update a channel member (e.g., change role)

DELETE /api/channels/{channelId}/members/{userId}

Remove a member from a channel

POST /api/channelmembers/bulk

Bulk add members to multiple channels
