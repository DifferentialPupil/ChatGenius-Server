Direct Message Endpoints

GET /api/directmessages/{directMessageId}

Get details of a specific direct message

POST /api/directmessages/{recipientId}

Send a new direct message to a user

GET /api/users/me

Get a list of the current user's direct message conversations

GET /api/directmessages/conversations/{userId}

Get messages in a direct message conversation with a specific user

PATCH /api/directmessages/{directMessageId}

Edit an existing direct message

DELETE /api/directmessages/{directMessageId}

Delete a direct message

POST /api/directmessages/{directMessageId}/reactions

Add a reaction to a direct message

DELETE /api/directmessages/{directMessageId}/reactions/{reactionId}

Remove a reaction from a direct message
