Mention Endpoints

GET /api/users/me/mentions

Get current user mentions

Description: Get a list of mentions for the current user

Query Parameters

GET /api/users/me/mentions/count

Count unread mentions

Description: Get a count of unread mentions for the current user

GET /api/mentions/{mentionId}

Get mention by ID

Description: Get details of a specific mention by ID

Path Parameters

PATCH /api/mentions/{mentionId}/read

Mark a mention as read

Description: Mark a specific mention as read

Path Parameters

PATCH /api/mentions/read-all

Mark all mentions as read

Description: Mark all mentions for the current user as read
