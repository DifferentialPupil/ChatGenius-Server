Workspace Endpoints

GET /api/workspaces

Get a list of all workspaces the user is a member of

GET /api/workspaces/me

Get details of the current user's primary workspace

GET /api/workspaces/{workspaceId}

Get details of a specific workspace

GET /api/workspaces/{workspaceId}/data

Get all data related to a workspace (workspace details, members, channels)

POST /api/workspaces

Create a new workspace

PATCH /api/workspaces/{workspaceId}

Update a workspace's details

GET /api/workspaces/{workspaceId}/users

Get all users in a workspace (Admin only)

POST /api/workspaces/{workspaceId}/users/invite

Invite users to a workspace (Admin only)

DELETE /api/workspaces/{workspaceId}/users/{userId}

Remove a user from a workspace (Admin only)

GET /api/workspaces/{workspaceId}/settings

Get workspace settings

PATCH /api/workspaces/{workspaceId}/settings

Update workspace settings

Valid values for aiGlobalAutonomyLevel: "manual", "review", "auto"
